## 2.	限定符（Quantifier）

### 2.1	\*  \+  ?

```
a*			a出现[0,∞]
a+			a出现[1,∞]
a?			a出现[0,1]
```



### 2.2	{    }

```
a{6}		a出现6次
a{2,6}		a出现2-6次
a{2,}		a出现2次以上
```



## 3.	或运算符（OR Operator）

```
(a|b)		匹配a或者b
(ab)|(cd)	匹配ab或者cd
```



## 4.	字符类（Character Classes）

```
[abc]		匹配a或者b或者c
[a-c]		同上
[a-fA-F0-9] 匹配a-f+A-F+0-9
[^0-9]		匹配非数字字符
```



## 5.	元字符（Meta-characters）

```
\d			匹配数字字符
\D			匹配非数字字符
\w			匹配单词字符（英文、数字、下划线）
\W			匹配非单词字符
\s			匹配空白字符（包含换行符、Tab）
\S			匹配非空白字符
.			匹配任意字符（换行符除外）
\b			标注字符的边界（全字匹配）
^			匹配行首
$			匹配行尾
```



## 6.	贪婪/懒惰匹配（Greedy / Lazy Match）

```
<.+>		默认贪婪匹配 任意字符
<.+?>		懒惰匹配 任意字符
```
