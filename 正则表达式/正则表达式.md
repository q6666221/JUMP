# 正则表达式
## 认识正则表达式
> 正则表达式就是更高级的**通配符**

## 元字符
| 元字符 | 说明                             |
|--------|----------------------------------|
| .      | 匹配出换行符以外的任意字符       |
| \w     | 匹配英数下划线，不包含中文日文等 |
| \s     | 匹配任意的空白字符               |
| \d     | 匹配数字                         |
| \b     | 匹配单词的开始或结束             |
| ^      | 匹配字符串的开始                 |
| $      | 匹配字符穿的结束                 |
